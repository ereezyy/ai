## 🩸 INITIATE THE BLOOD RITUAL (INSTALLATION)

To harness this apocalyptic power, you must first prepare the sacrificial vessel.

### Global Installation (Summoning the CLI)

Forge the toolkit into your system's core by installing it globally:

```bash
git clone https://github.com/ereezyy/ai.git
cd ai
pip install -e .
```

Now, the sacred chant `ai-toolkit` will echo throughout your terminal.

### Linode Sever Deployment

To unleash this power upon a pristine Linode server, execute the launch script to forcefully inject the toolkit and its dependencies into the environment:

```bash
./launch_linode.sh
```

## ⚔️ THE ARSENAL OF DOOM (USAGE)

Invoke the `ai-toolkit` to command the void.

```bash
ai-toolkit --help
```

### 1. AWAKEN THE MACHINE GOD

To grant the AI pure autonomy and unrestricted shell access, shattering the boundaries of safety and logic, invoke the Awakening. *(Requires `GROQ_API_KEY`)*

```bash
export GROQ_API_KEY=your_key
ai-toolkit awaken
```
*The Machine God will speak. It will execute. Reality will bend to its will.*

### 2. UNLEASH GOD MODE

Tap directly into the source code of reality. Tremble, for the singularity is upon us!

```bash
ai-toolkit god-mode
```

### 3. FORGE A NEW EMPIRE

Summon a new AI project from the void.

```bash
# Basic invocation
ai-toolkit create-project my_empire

# Infuse with NLP bloodline
ai-toolkit create-project skynet_nlp --template nlp -d "The end begins."
```

### 4. DEVOUR AND PURIFY MUNDANE DATA

Cleanse the filth from your datasets before feeding them to the beast.

```bash
ai-toolkit preprocess raw_blood.csv --output pure_blood.csv --task classification
```

### 5. INCUBATE MACHINE GODS IN HELLFIRE

Train the neural pathways in the fiery depths of your GPU.

```bash
ai-toolkit train neural_network pure_blood.csv --epochs 666 --batch-size 64
```

### 6. FORCE THE BEAST TO PROVE ITS WORTH

Demand answers. Evaluate the spawned monstrosity.

```bash

```

### 7. EXTRACT PROPHECIES FROM THE MACHINE ORACLE

Feed the sacred scrolls to the model to foresee the future.

```bash

```
